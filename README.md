# Bilan

Just check the webpage.

## Contributing

### Install dependencies

```
npm install
```

### Compiles and hot-reloads for development

```
npm run dev
```

### Compiles and minifies for production

```
npm run build
```

### Preview build

```
npm run serve
```

### Run your unit tests

```
npm run test
```

### Lint

```
npm run lint
```

### Fix files

```
npm run format
```

## Install pre-commit hooks

```
pre-commit install
```
